## Welcome to Zzyra API

Zzyra's API provides programmatic access to all platform capabilities, enabling you to build custom integrations, automate workflows, and extend the platform's functionality.

<Alert type='info'>
  **API Version**: v1.0.0 | **Base URL**: `https://api.zzyra.com/v1` | **Rate
  Limit**: 1000 requests/minute
</Alert>

## Quick Start

<div className="api-quickstart">
  <div className="quickstart-tabs">

curl -H "Authorization: Bearer YOUR_API_KEY" \

```javascript
const response = await fetch('https://api.zzyra.com/v1/workflows', {
  method: 'POST',

    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'My Automation',
    description: 'Automated trading strategy',
    blocks: [
      {
        type: 'price_monitor',
        config: {
          symbol: 'ETH',
          threshold: 3000
        }
      }
    ]
  })
});
```
      </Tab>
      
      <Tab title="WebSocket Events">
```javascript
const ws = new WebSocket('wss://api.zzyra.com/v1/events');

ws.onmessage = (event) => {
const data = JSON.parse(event.data);
console.log('Workflow executed:', data);
};

// Subscribe to workflow events
ws.send(JSON.stringify({
type: 'subscribe',
events: ['workflow.executed', 'workflow.failed']
}));

````
      </Tab>
    </Tabs>
  </div>
</div>

## Authentication

Zzyra uses API keys for authentication. Include your API key in the `Authorization` header with all requests.

<div className="auth-examples">
  <Tabs>
    <Tab title="API Key">
```bash
curl -H "Authorization: Bearer sk_live_..." \
  https://api.zzyra.com/v1/workflows
````

    </Tab>

    <Tab title="OAuth 2.0">

```bash
# Get access token
curl -X POST https://api.zzyra.com/oauth/token \
  -d "grant_type=client_credentials" \
  -d "client_id=YOUR_CLIENT_ID" \
  -d "client_secret=YOUR_CLIENT_SECRET"

# Use access token
curl -H "Authorization: Bearer ACCESS_TOKEN" \
  https://api.zzyra.com/v1/workflows
```

    </Tab>

    <Tab title="Web3 Wallet">

```javascript
// Connect wallet and sign message
const signature = await wallet.signMessage("Authenticate with Zzyra API");
const response = await fetch("https://api.zzyra.com/v1/auth/wallet", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    address: wallet.address,
    signature: signature,
  }),
});
```

    </Tab>

  </Tabs>
</div>

## WebSocket Events

<div className="websocket-events">
  <div className="event-types">
    <CardGroup cols={2}>
      <Card title="Workflow Events" icon="workflow">
        <ul>
          <li><code>workflow.created</code> - New workflow created</li>
          <li><code>workflow.updated</code> - Workflow configuration changed</li>
          <li><code>workflow.deleted</code> - Workflow removed</li>
          <li><code>workflow.executed</code> - Workflow execution started</li>
          <li><code>workflow.completed</code> - Workflow execution finished</li>
          <li><code>workflow.failed</code> - Workflow execution failed</li>
        </ul>
      </Card>
      
      <Card title="System Events" icon="cog">
        <ul>
          <li><code>system.maintenance</code> - Scheduled maintenance</li>
          <li><code>system.update</code> - Platform update</li>
          <li><code>rate_limit.warning</code> - Approaching rate limit</li>
          <li><code>security.alert</code> - Security-related events</li>
        </ul>
      </Card>
    </CardGroup>
  </div>
  
  <div className="websocket-example">
```javascript
const ws = new WebSocket('wss://api.zzyra.com/v1/events');

// Subscribe to specific events
ws.onopen = () => {
ws.send(JSON.stringify({
type: 'subscribe',
events: ['workflow.executed', 'workflow.completed', 'workflow.failed']
}));
};

// Handle incoming events
ws.onmessage = (event) => {
const data = JSON.parse(event.data);

switch (data.type) {
case 'workflow.executed':
console.log('Workflow started:', data.workflow_id);
break;
case 'workflow.completed':
console.log('Workflow completed:', data.workflow_id);
break;
case 'workflow.failed':
console.log('Workflow failed:', data.workflow_id, data.error);
break;
}
};

```
  </div>
</div>
